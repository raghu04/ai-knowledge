# AI Knowledge

This repository is a concise collection of practical notes and best practices for working effectively with AI tools, especially Claude.

## What this repository contains

- Guidance for getting better continuity on the Claude Free plan
- Prompt patterns for resuming long tasks without losing context
- A simple knowledge base for AI-assisted workflows and productivity

## Featured document

- [Claude Free Plan Best Practices](Claude/Claude-Free-Plan-Best-Practices.md)

## Key ideas

- Keep the same chat whenever possible
- Prefer “Continue” over “Retry” when a response is cut off
- Ask for checkpoints and milestone-based progress updates
- Save important outputs locally so work is not lost
- Keep prompts short, specific, and focused on the next step

## Suggested use

1. Read the best-practices guide.
2. Copy the prompt templates that fit your workflow.
3. Adapt the patterns to your own projects and AI tools.

This repo serves as a simple reference for more productive and reliable AI-assisted work.
